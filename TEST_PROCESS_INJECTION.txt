## STEP 6: TROUBLESHOOTING

### If No Processes Show:
1. Check RAT client is running and connected
2. Verify WebSocket connection (bridge logs show "Client connected")
3. Run RAT as Administrator: `runas /user:Administrator rat_client_injection.exe`

### If Injection Fails:
1. **Run as Administrator** (required for process injection)
2. **Check DLL path**: Use absolute path
3. **Match architectures**: x86 DLL → x86 process, x64 DLL → x64 process
4. **Avoid protected processes**: Don't inject into system processes
5. **Test with safe processes first**: notepad.exe, calc.exe, mspaint.exe

### Common Errors:
- **"Access denied"**: Run RAT as Administrator
- **"DLL not found"**: Use absolute path to DLL
- **"Invalid process"**: Process may have terminated
- **"Failed to allocate memory"**: Target process may be protected
